pyssh - простой враппер SSH-клиента позволяющий указывать порт в адресе сервера:
   $ pyssh user@host:3045

scinit - облегчает использование смарткарт в связке с SSH-агентом, примеры:
   Инициализировать SSH-агент
   Установить переменные окружения
   Запустить ssh
   $ scinit load:exec ssh -A user@host

   Переинициализировать SSH-агент
   $ scinit reload

   Переинициализировать SSH-агент и запустить шелл с установлеными переменными окружения
   $ scinit reload:shell

install-keys - генерирует ~/.ssh/authorized_keys следующим образом:
   1. Берет все ключи из ~/.authorized_keys.d/*
   2. Скачивает все указанные в командной строке URL
   3. Сортирует, убирает повторы
   4. Записывает результат

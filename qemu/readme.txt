Простой враппер для QEMU, позволяет запускать ВМ ре разбираясь в
куче параметров просто написав минимальную конфигурацию

runvm - собственно скрипт запуска
qcow-compact - скрипт для сжатия образов qcow2
config.sample - максимально подробное описание конфигурации ВМ
any-linux - простая конфигурация для запуска любого сколь-нибудь распространенного Linux

Изначально писалось для MacOS (ARM и x86), также поддерживается Linux
